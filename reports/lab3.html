Розробка макету

У своєму додатку я вирішив використати Flexbox макет. Це є потужним модулем CSS, який забезпечує простий та ефективний спосіб розташування елементів в контейнері, незалежно від їх розміру та кількості. Завдяки цьому, я можу створити макет сайту, який реагуватиме на зміну розмірів екрану та забезпечить гнучкість в організації вмісту мого веб-сайту.

Висновок з лабораторної роботи

У цій лабораторній роботі ми зосередилися на верстці HTML-документу, використанні блокової верстки та засобах CSS і Flexbox. Під час виконання роботи ми набули практичних навичок створення структури веб-сторінок за допомогою HTML, розуміння ролі блокової верстки в організації вмісту сторінок та використання CSS для оформлення цього вмісту.

Особливу увагу ми приділили Flexbox, потужному інструменту CSS для створення гнучких та адаптивних макетів веб-сторінок. Вивчення Flexbox дозволило нам зрозуміти, як ефективно розташовувати та організовувати елементи на сторінці, забезпечуючи їх адаптивне розміщення на різних пристроях і екранах.

Крім того, ми також дослідили можливості CSS для стилізації тексту, введеного користувачем у поля вводу. Це важливий аспект веб-дизайну, оскільки дозволяє покращити взаємодію користувача з веб-сайтом та зробити введений текст більш помітним і зрозумілим.

У результаті виконання лабораторної роботи ми отримали цінний досвід у створенні та стилізації веб-сторінок, який буде корисним для подальшого розвитку навичок у веб-розробці.
