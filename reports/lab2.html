Способи підключення стилів

У своєму додатку я використав вбудовані та зовнішні стилі.

  1. Вбудовані стилі: 
    
    <h3 style="font-weight: bold; margin: 0.5em 0">Кошик</h3>
  2. Зовнішні стилі: 
    
    <link rel="stylesheet" href="css/style.css">

    У файлі 'styles.css':
    header {
            display: flex;
            flex-direction: row;
            align-items: center;
            justify-content: space-around;
    }

Селектори тегу:

Приклад використання:

<header>...</header>

header { height: 130px; width: 100%; background-color: #FFFFFF; top: 0; left: 0; right: 0; font-family: sans-serif; }

Селектори класу:

Приклад використання:


<div class="top-btn-wrapper">
  <div class="buttons">
     <a href="">Робота</a>
     <a href="">Питання-відповіді</a>
     <a href="">Вхід</a>
  </div>
</div>
.top-btn-wrapper { display: flex; justify-content: flex-end; height: 20px; top: 0; left: 0; right: 0; width: 100%; }

Селектори ідентифікаторів:

Приклад використання:

<img id="company-logo" src="img/header/logo.png" alt=""><br>

#company-logo { width: 100px; height: 100px; }

Інші селектори:

Приклад використання:

  1. Універсальний селектор:
    * { font-family: sans-serif; }
  2. Дочірні селектори:
    .search-form > input { flex: 1; border: none; outline: none; background-color: #F5F5F5; padding: 0 15px; height: 30px; }

CSS: Шрифти Текст Таблиці Фон Контур Списки СSS Просунутий

  1. Шрифти:
    * { font-family: sans-serif; }
  2. Фон:
    main { background-color: #F8F8F8; }
  3. Таблиці:
    .categories-wrapper { display: flex; flex-wrap: wrap; width: 650px; margin: 20px auto; justify-content: space-between; }
    .category-wrapper { display: flex; flex-direction: column; width: 200px; height: 200px; background-color: white; box-shadow: 0 0 10px rgba(0, 0, 0, 0.1); margin: 5px; }
  4. Колір тексту:
    width: 400px; height: 43px; align-items: center; color: #ececec; background-color: #F5F5F5; border-radius: 10px; }

Висновок з лабораторної роботи

У ході виконання лабораторної роботи з теми "Каскадні таблиці стилів, селектори, ідентифікатори, стильове оформлення текстових елементів в HTML-документах" я здобув значні практичні навички у використанні CSS для оформлення веб-сторінок.

Починаючи з основних концепцій селекторів та ідентифікаторів, я освоїв різноманітні методи вибору елементів на сторінці для застосування до них стилів. Використання селекторів дозволило мені точно визначити, які елементи сторінки я хочу стилізувати, що дозволило досягти більшої гнучкості та точності у вигляді сторінки.

Під час виконання роботи я також вивчив широкий спектр властивостей CSS для кольору, фону, відступів, а також плаваючих елементів. Це дозволило мені створювати більш привабливий та естетично збалансований вигляд веб-сторінок. Навички роботи з внутрішніми та зовнішніми відступами дозволили мені керувати простором навколо елементів, щоб досягти оптимального розташування та міжелементного простору.
